ТЕСТОВОЕ ЗАДАНИЕ
Unity Developer (удаленно)

	* Задание должно быть выполнено полностью (ВСЕ описанные пункты).
	* Помимо работоспособности будут оцениваться качество и чистота кода.
	* Не использовать готовые решения с Unity Asset Store, содержащие С# код, кроме
допустимых, описанных в данном ТЗ. Все сделать самостоятельно с нуля.
	* Можно использовать бесплатный аг{ вроде моделей героев и анимаций перемещений.

1. В Unity 2021.3.* (последней LTS-версии) создать 30 сцену с:
	* Р1апе-полом и разбросанными по нему примитивами для имитации локации,
	* камерой от 3-его лица (с управлением обзором мышью по горизонтали и вертикали без инверсии),
	* Direction Light

2. Разместить на сцене зрамип ро!п!$ для механики случайного выбора точки Spawn игрока (в каждой точке появляется только один игрок).

3. Сделать multiplayer, используя Mirror Networking:
	* Организовать создание комнаты и подключение к ней.

	* После подключения к комнате для каждого клиента должен создаваться управляемый персонаж, способный перемещаться по WASD в плоскости пола относительно позиции камеры.

4. У каждого игрока есть способность - Рывок - резкое перемещение-импульс в сторону движения на определённое расстояние. Использование по inputKey=LMB.

	* Должна существовать настройка, позволяющая регулировать расстояние в ефйог.

5. Если игрок задевает другого игрока Рывком, задетый на Х=3 секунды меняет свой цвет.

	* Должна существовать настройка, позволяющая регулировать длительность смены цвета в editor.

6. В состоянии измененного цвета игрок неуязвим и не реагирует на Рывки противников.

7. Первый игрок, Х=3 раза попавший Рывком по противникам (нужна индикация счета каждого игрока), выигрывает игру:

	а. выводится имя победителя;
	Ь. через Х=5 секунд матч перезапускается;
	с. все игроки появляются в случайных точках;
	d. сбрасывается счет каждого игрока.
